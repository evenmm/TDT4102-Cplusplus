
Når vi bruker pekere må vi skille mellom å endre minneaddressen som pekeren henviser
til, og innholdet som blir pekt på. For å endre innholdet som pekes på gjør man
følgende:
void foo(double *b){
*b = 2.0;
}
For å endre minneaddressen som blir pekt på gjør man følgende:
void foo(double *a, double *b){
b = a;
}
